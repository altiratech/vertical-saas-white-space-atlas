# Vertical SaaS White-Space Atlas Founding Packet

## Mission

Turn public business-demography and labor data into a ranked answer to:
- where to build
- where to sell
- where to acquire
- what to ignore

## Why this exists

Most vertical software and buy-and-build decisions still rely on anecdotes:
- "this industry is fragmented"
- "there is lots of paperwork"
- "nobody serves this niche well"

This product should replace hand-waving with a public-data-backed ranked move engine.

## Core user

The first user is a founder/operator choosing which market to enter or target next.

## Core output

Each industry row should end in:
- a recommended move
- visible evidence
- caveats
- a confidence score

## First source families

- Census County Business Patterns
- SBA size standards
- BLS employment / projection files

Optional later layers:
- O*NET
- geographic expansion
- occupation overlays

## First loop

1. build a canonical industry table
2. score software wedge and roll-up wedge quality
3. surface ranked industries
4. explain the ranking in plain language

## Success definition for v1

The user can look at 20-50 ranked industries and say:
- these are clearly better first targets than those
- I understand why
- I can act on this now

## Explicit defers

Do not treat these as v1 requirements:
- account-level data
- workflow-level benchmarking by company
- live CRM workflows
- proprietary spend or buyer data
